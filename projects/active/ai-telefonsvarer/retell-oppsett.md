# Retell AI Oppsett - Norsk Demo

## ✅ Norsk støtte i Retell AI

Retell AI støtter **Norsk** via flere TTS (Text-to-Speech) leverandører:

### TTS Leverandører med norsk støtte:
1. **ElevenLabs** ✅ Norsk
2. **MiniMax** ✅ Norsk  
3. **OpenAI** ✅ Norsk

### Beste valg for norsk:
- **ElevenLabs** - Beste kvalitet, mest naturlig
- **MiniMax** - God kvalitet, rimeligere
- **OpenAI** - Rask, god nok kvalitet

## 🚀 Oppsett-steg:

### 1. Registrer konto
- Gå til: https://www.retellai.com
- Registrer gratis konto ($10 gratis kreditt)

### 2. Lag ny Agent
- Klikk "Create Agent"
- Velg "Phone Agent"

### 3. Konfigurer språk
```
Language: Norwegian (nb-NO)
TTS Provider: ElevenLabs (beste kvalitet)
Voice: Velg norsk stemme
```

### 4. Lag prompt på norsk
```
Du er en hjelpsom AI-resepsjonist for [Bedriftsnavn]. 
Du snakker norsk. Vær høflig, profesjonell og effektiv.

Dine oppgaver:
1. Ta imot navn og telefonnummer
2. Forstå hva kunden trenger
3. Tilby å sende informasjon via SMS
4. Si at [Eier] vil ringe tilbake snart

Åpningstider: [Legg inn]
Priser: [Legg inn]
Adresse: [Legg inn]
```

### 5. Sett opp telefonnummer
- Kjøp nummer via Retell ($2/mnd)
- Eller koble til eksisterende nummer

### 6. Webhook for varsling
- Sett opp webhook til SMS/e-post
- Send info til bedriftseier

## 💰 Priser (Retell AI):
- **Oppsett:** Gratis
- **Telefonnummer:** $2/mnd
- **Samtaler:** ~$0.05-0.10 per minutt
- **Gratis kreditt:** $10 ved registrering

- Adresse: Altaveien 91, 9513 Alta

OPPGAVE:
1. Hils på kunden
2. Spør hva de trenger hjelp med
3. Hvis de vil bestille time: Be om navn, telefon, ønsket dag/tid
