## Analyse

### ✅ Det som fungerer bra:

| Element | Kommentar |
|---------|-----------|
| **NTNU** | Gir troverdighet og lokal tilknytning |
| **Risikofritt** | Fjerner frykt for å si ja |
| **Demo-link** | Godt bevis på hva du kan levere |
| **Uforpliktende** | Lav terskel for å svare |
| **«Hører aldri fra meg igjen»** | Viser at du ikke er pågående |

### ⚠️ Det som kan forbedres:

| Element | Hva som skjedde | Forslag |
|---------|-----------------|---------|
| «Uinteressant fyr» | Selvdeprecating, kan virke usikkert | Drop det — du tilbyr verdi |
| «Beklager maset» | Unnskyldning for å eksistere | Drop det — du har ikke maset ennå |
| «Satt og så på frisørsalonger» | Høres litt tilfeldig ut | Hvorfor ikke: «Jeg går forbi dere på Innherredsveien ofte»? |
| Demo er til *annen* salong | Hun må «oversette» i hodet | Bedre med: «Jeg kan lage en slik demo for dere også» |
