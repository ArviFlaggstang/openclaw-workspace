# Nettside-prosess for Altaie Frisør (Demo)

## Teknologi-stack
- **Framework:** Next.js (React) eller ren HTML/CSS/JS
- **Styling:** Tailwind CSS (raskt, moderne)
- **Hosting:** Vercel (gratis) eller Netlify (gratis)
- **Domene:** .no domene hos Domeneshop (149 kr/år)
- **Bilder:** Unsplash (gratis) eller kundens egne bilder

## Steg-for-steg prosess

### 1. Design & Utvikling (2-4 timer)
- [ ] Sette opp prosjekt
- [ ] Lage layout (header, hero, tjenester, om oss, kontakt, footer)
- [ ] Legge til innhold (tekst + bilder)
- [ ] Mobiltilpasning
- [ ] Teste

### 2. Deploy (15 min)
- [ ] Koble til GitHub
- [ ] Deploy til Vercel/Netlify
- [ ] Få gratis .vercel.app eller .netlify.app domene

### 3. Domene (valgfritt, 15 min)
- [ ] Kjøpe .no domene hos Domeneshop
- [ ] Koble til hosting

### 4. Overlevering (30 min)
- [ ] Vise kunden
- [ ] Gjøre endringer
- [ ] Betaling
- [ ] Overføre eierskap

## Demo for Mohammed

Vi lager en **Basis**-versjon som viser:
1. Hero-seksjon med bilde av salongen
2. Tjenester (herreklipp, barneklipp, barbering)
3. Om Mohammed / historien
4. Åpningstider & kontakt
5. Facebook/Instagram lenker

**Tid:** 2-3 timer
**Kostnad for oss:** 0 kr (bruker gratis verktøy)
**Potensiell inntekt:** 8 000-15 000 kr

## Fordeler med denne stacken

- **Gratis hosting** = høy margin for oss
- **Rask utvikling** = flere kunder per uke
- **Moderne teknologi** = ser proft ut
- **Enkelt å vedlikeholde** = passiv inntekt
