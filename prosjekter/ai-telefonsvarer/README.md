# AI Telefonsvarer for Bedrifter

## Konsept
En AI-drevet telefonsvarer som tar imot samtaler når bedriften er opptatt eller stengt.

## Hva den gjør:
1. Tar imot samtalen når bedriften ikke svarer
2. Forstår hva kunden vil (bestille time, spørre om pris, etc.)
3. Tar notater og sender SMS/e-post til bedriften
4. Kan gi grunnleggende info (åpningstider, priser, adresse)
5. Integrerer med booking-systemer

## Teknologi:
- **VAPI** eller **Bland AI** for stemme-AI
- **Twilio** for telefonnummer og samtaler
- **OpenAI** for språkforståelse
- **Webhook** til bedriftens systemer

## Pris for bedrifter:
- **Oppsett:** 2000-5000 kr
- **Månedlig:** 500-1000 kr (avhengig av samtalevolum)

## Fordeler for bedrifter:
- Aldri miste kunder som ringer
- 24/7 tilgjengelighet
- Automatisk booking
- Profesjonell fremtoning
- Språkstøtte (norsk, engelsk, etc.)
