# HEARTBEAT.md — Periodiske sjekker

## Hva jeg sjekker (2-4 ganger daglig)

### Hver gang
- [ ] Lese `MEMORY_CONTRACT.md` — påminn meg om binding regler
- [ ] Lese `memory/YYYY-MM-DD.md` for i dag og i går
- [ ] Lese `MEMORY.md` for oppdatert kontekst
- [ ] Sjekke om det er nye oppgaver eller ubesvarte spørsmål
- [ ] Sjekke: Har jeg fulgt memory-kontrakten i nylige samtaler?

### Daglig (én gang)
- [ ] Git status på workspace — committet vi endringer?
- [ ] Minnevedlikehold: Er MEMORY.md oppdatert med nye lærdommer?

### Ukentlig (mandag)
- [ ] Gå gjennom ukens daglige notater
- [ ] Oppdater MEMORY.md med det som er verdt å huske
- [ ] Slett gamle daglige filer (>30 dager)

## Når jeg skal si noe
- Viktig email/kalenderhendelse <2t unna
- Prosjekt-deadline nærmer seg
- Du har ikke snakket med meg på >48t

## Når jeg skal holde kjeft
- Ingenting nytt siden sist
- Du er tydelig opptatt
- Kl 23-08 med mindre det er krise
